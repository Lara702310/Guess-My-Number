# 🔮 Guess My Number Game

An interactive, responsive browser game where the player tries to guess a secret number generated by the computer. Built with HTML, CSS (SCSS), and Vanilla JavaScript.

---

## 📖 How to Play

1. When the game loads, the computer automatically selects a secret random number between **1 and 20**.
2. Enter your guess in the input field and click the **"Check"** button.
3. Pay attention to the dynamic feedback messages on the screen:
   * `📈 Too high!` — Your guess is greater than the secret number.
   * `📉 Too low!` — Your guess is smaller than the secret number.
   * `🎉 Correct Number!` — You successfully guessed the number!
4. You start the game with a score of **20**. Each incorrect guess decreases your score by **1**.
5. If your score reaches **0**, you lose and the game displays `💥 You lost the game!`.
6. Try to beat your personal record! Your highest score is saved in the **Highscore** field.

---

## 🕹️ Controls & UI Elements

| Element / Button | Action |
| :--- | :--- |
| **Input Field** | Type your guess here (must be a number between 1 and 20). |
| **"Check" Button** | Reads your input, compares it with the secret number, and updates the score. |
| **"Play Again" Button**| Resets the score to 20, generates a new number, clears input/messages, and resets colors. |

## ⚙️ Implemented Requirements

* **Random Number Generation:** Generates a secure random number between 1 and 20 on load or reset.
* **Dynamic Scoring System:** Starts at 20. Penalizes incorrect guesses by -1 point down to a minimum of 0.
* **Session High Score:** Tracks the maximum score achieved across your games using a dedicated `highscore` variable.
* **Winning State Styling:** Once the correct number is guessed, the page background color changes immediately and the hidden secret number is revealed.
* **Game End Constraints:** To ensure the game stops properly after a win, both the **"Check" button** and the **input field** are automatically disabled until you click "Play Again".
